1. 显示 config 的配置

   - git config --list

2. 配置用户信息

   - git config user.name 'xxx'
   - git config user.name 'xxx@xxx.com'

3. git config --scope 作用域设置

   - local 只对某个仓库有效
   - global 对当前用户所有仓库有效
   - system 对系统登录的所有用户有效
   
4. 清除配置

   git config  --local --unset user.name
